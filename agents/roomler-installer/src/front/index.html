SystemContext mode — one elevated step left:

The perMachine MSI is installed and the agent is enrolled. To turn on remote control of the Windows lock screen, UAC dialogs, and admin apps, open an elevated PowerShell (right-click → Run as administrator) and run:

roomler-agent set-service-env-var --name ROOMLER_AGENT_ENABLE_SYSTEM_SWAP --value 1
roomler-agent restart-service

The wizard can't write to HKLM\SYSTEM\…\Environment directly because the wizard process inherits a non-elevated token from your double-click. Full automation (a WiX custom action gated on ENABLE_SYSTEM_CONTEXT=1 set by the wizard at msiexec time) is the next-cycle slice.
