SystemContext mode active.

The perMachine MSI is installed, the daemon is enrolled, and the SCM service runs with ROOMLER_AGENT_ENABLE_SYSTEM_SWAP=1 in its Environment block. Remote control can now drive the Windows lock screen, UAC dialogs, and admin apps when an operator connects from the browser.

To disable SystemContext mode later, run roomler-agent disable-system-context from an elevated shell, or re-run this wizard and pick the attended-service role.
